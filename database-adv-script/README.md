# Airbnb Advanced SQL Project

This project demonstrates advanced SQL querying, performance optimization, indexing, and partitioning for a simulated Airbnb database.

## Files
- joins_queries.sql : INNER, LEFT, FULL OUTER JOIN examples
- subqueries.sql : Correlated & non-correlated subqueries
- aggregations_and_window_functions.sql : COUNT, GROUP BY, RANK
- database_index.sql : Index creation
- perfomance.sql : Complex query + optimized version
- partitioning.sql : Table partitioning
- index_performance.md : Indexing report
- optimization_report.md : Query optimization report
- partition_performance.md : Partitioning report
- performance_monitoring.md : Monitoring & refinement

## How to run
Open your SQL client, execute the files in order, and verify outputs.
